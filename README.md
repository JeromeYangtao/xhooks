# xhooks

xhooks 是一个 React Hooks 库，致力提供常用且高质量的 Hooks。

## ✨ 亮点

- 容易使用
- 包含大量具有高级功能的 hooks
- 基础 hooks 不再重复开发
- 使用 TypeScript 开发，具有完备的类型提示

- [ ] useFusionDialog

## 📦 按照

```
npm i xhooks --save
```

## 🔨 使用

```typescript
import { uesDialog } from "xhooks";
```

## 🤝 贡献

```
$ git clone https://github.com/JeromeYangtao/xhooks.git
$ cd xhooks
$ npm run init
$ npm start
```

打开浏览器和访问 http://127.0.0.1:8001
